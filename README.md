Purpose of the App:

the purpose of CYBERMONK is to help users study and verify life hacks with flashcards. The app will present questions, 
loop through each hack, and provide a total score based on the user's ability to spot a myth.

Design Considerations:

The application was developed for Android devices using Android Studio. The app was built using Kotlin Designed the User Interface using XML layouts in Android Studio.
